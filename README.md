# Gestionnaire de Budget Personnel | Gerenciador de Orçamento Pessoal

## 🇫🇷 Français

### Description
Une application web légère et intuitive permettant de suivre ses revenus et dépenses. Ce projet a été réalisé dans le cadre du **TP1** pour démontrer des compétences en développement front-end moderne (HTML5, CSS3, JavaScript ES6).

### Fonctionnalités principales
* **Résumé en temps réel (GB-01) :** Affichage automatique du total des revenus, des dépenses et du solde net.
* **Gestion des transactions (GB-03, GB-04) :** Ajout et suppression de transactions avec confirmation de sécurité.
* **Persistence des données (GB-06, GB-07) :** Importation et exportation des données au format JSON.
* **Design Adaptatif (GB-08) :** Interface entièrement responsive, optimisée pour mobile (testée sur iPhone 12 Pro) et desktop.
* **Chargement Dynamique :** Utilisation de l'API `fetch()` pour charger les données initiales depuis un dossier structuré.

### Structure du Projet
* `/data` : Contient le fichier `data.json` pour le chargement initial.
* `index.html` : Structure sémantique et accessible.
* `style.css` : Design moderne utilisant Flexbox et Media Queries.
* `script.js` : Logique métier et manipulation du DOM.

## 🇧🇷 Português

### Descrição
Uma aplicação web leve e intuitiva para monitorar receitas e despesas. Este projeto foi desenvolvido como parte do **TP1** para demonstrar competências em desenvolvimento front-end moderno (HTML5, CSS3, JavaScript ES6).

### Principais Funcionalidades
* **Resumo em Tempo Real (GB-01):** Exibição automática do total de receitas, despesas e saldo líquido.
* **Gestão de Transações (GB-03, GB-04):** Adição e remoção de transações com confirmação de segurança.
* **Persistência de Dados (GB-06, GB-07):** Importação e exportação de dados em formato JSON.
* **Design Adaptativo (GB-08):** Interface totalmente responsiva, otimizada para mobile (testada no iPhone 12 Pro) e desktop.
* **Carregamento Dinâmico:** Uso da API `fetch()` para carregar dados iniciais a partir de uma pasta estruturada.

### Estrutura do Projeto
* `/data`: Contém o arquivo `data.json` para o carregamento inicial.
* `index.html`: Estrutura semântica e acessível.
* `style.css`: Design moderno utilizando Flexbox e Media Queries.
* `script.js`: Lógica de negócio e manipulação do DOM.
